# OpenVPN-WEB

Web portal for OpenVPN gateway. Provides some reporting information and basic data visualization for usage of the OpenVPN server.

# Installation
Copy all of the files to the server, ensure correct execution rights are given to the files. Then launch the install.php inside your browser
